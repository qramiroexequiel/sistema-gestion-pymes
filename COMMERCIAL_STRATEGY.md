# Estrategia Comercial - Custom Internal Management System

## 1. Estrategia de Precios

### 1.1 Precio Público vs. Precio Final

**Precio Público (Upwork / Propuestas Iniciales):**
- **Setup inicial:** "Starting from $1,200 USD"
- **Mantenimiento mensual:** "Starting from $200 USD/month"

**Razón:** "Starting from" permite flexibilidad sin comprometer el precio base. El cliente entiende que puede haber variaciones según necesidades específicas.

**Precio Final (Negociación):**
- **Rango aceptable setup:** $1,200 - $1,800 USD
- **Rango aceptable mantenimiento:** $200 - $300 USD/mes

**Cuándo subir el precio:**
- Cliente requiere migración de datos masiva (>500 registros)
- Necesita personalizaciones específicas fuera del alcance estándar
- Requiere capacitación adicional extensa (>4 horas)
- Cliente en industria con requisitos regulatorios complejos
- Cliente solicita implementación urgente (<7 días)

**Cuándo mantener o bajar ligeramente:**
- Cliente con volumen bajo de datos (<100 registros)
- Cliente que acepta implementación en 14+ días
- Cliente que se compromete a 6+ meses de mantenimiento
- Cliente que puede proporcionar datos bien organizados

**Regla de oro:** Nunca bajar de $1,000 USD en setup ni de $180 USD/mes en mantenimiento. Si el cliente no puede pagar esto, no es el cliente ideal.

## 3. Mantenimiento Mensual - Definición Exacta

### 3.1 Qué SÍ Incluye

**Soporte Técnico:**
- Resolución de consultas técnicas vía email
- Asistencia para problemas del sistema
- Ayuda con configuración de usuarios y permisos
- Soporte para uso del sistema (cómo hacer X)
- Horas incluidas según paquete (ver sección 2)

**Actualizaciones del Sistema:**
- Parches de seguridad (automáticos, sin costo adicional)
- Actualizaciones menores de funcionalidad estándar
- Corrección de bugs reportados
- Mejoras de rendimiento del sistema

**Backups y Seguridad:**
- Backup diario automático de la base de datos
- Retención de backups por 30 días
- Monitoreo básico de seguridad y disponibilidad
- Restauración de backups en caso de necesidad (dentro de horas de soporte)

**Mejoras Menores (según paquete):**
- Ajustes de campos existentes (agregar/quitar campos en formularios)
- Modificaciones menores en reportes (cambiar columnas, agregar filtros básicos)
- Cambios en permisos y roles
- Ajustes de configuración por empresa
- **Límite:** Mejoras que requieren menos de 2 horas de desarrollo

### 3.2 Qué NO Incluye

**Desarrollo de Nuevas Funcionalidades:**
- Nuevos módulos completos
- Integraciones con sistemas externos (APIs, facturación electrónica, etc.)
- Automatizaciones complejas
- Nuevas funcionalidades significativas fuera del alcance estándar

**Migraciones Masivas Adicionales:**
- Importación de datos después de la implementación inicial
- Migración desde otros sistemas
- Limpieza masiva de datos

**Capacitación Adicional Extensiva:**
- Sesiones de capacitación adicionales (más allá de las incluidas)
- Capacitación para nuevos empleados (se puede hacer con documentación)
- Consultoría estratégica extensa (solo en Premium con límite de horas)

**Servicios de Terceros:**
- Hosting adicional (si el cliente requiere servidor dedicado)
- Dominios personalizados
- Certificados SSL personalizados (si se requiere configuración especial)

**Regla de oro:** Si una solicitud requiere más de 2 horas de desarrollo o es claramente una nueva funcionalidad, se cotiza por separado.

### 4.3 Respuesta a Objeciones Típicas

**Objeción: "Es caro"**

**Respuesta:**
Entiendo su preocupación. Permítame mostrarle el valor real:

- **Setup de $1,500:** Incluye sistema completo configurado, migración de datos, capacitación y garantía. Un desarrollo desde cero costaría $5,000-10,000 USD y tomaría meses.

- **Mantenimiento de $250/mes:** Incluye soporte técnico, actualizaciones de seguridad, backups diarios, y mejoras menores. Un desarrollador freelance cobra $50-100 USD/hora. Con 4 horas incluidas, ya está ahorrando.

- **ROI:** Si el sistema le ahorra 5 horas/semana en tareas administrativas, se paga solo en menos de un mes.

**Alternativa:** Podemos empezar con el paquete Basic ($1,200 setup, $200/mes) y escalar cuando crezca.

---

**Objeción: "Puedo hacerlo con Excel"**

**Respuesta:**
Excel es excelente para análisis, pero tiene limitaciones para gestión operativa:

- **Multi-usuario:** Excel no maneja bien múltiples usuarios editando simultáneamente. Este sistema sí.

- **Seguridad:** ¿Quién tiene acceso a su archivo Excel? ¿Dónde está respaldado? Este sistema tiene roles, permisos y backups automáticos.

- **Escalabilidad:** Con 50+ clientes y 100+ operaciones/mes, Excel se vuelve lento y propenso a errores. Este sistema maneja miles de registros sin problemas.

- **Tiempo:** ¿Cuántas horas/semana dedica a mantener sus Excel actualizados? Este sistema automatiza eso.

**Puede seguir usando Excel para análisis**, pero el sistema le da la base de datos organizada y segura que Excel no puede proporcionar.

---

**Objeción: "¿Qué pasa con mis datos?"**

**Respuesta:**
Sus datos son 100% suyos. Esto no es marketing, es técnicamente así:

- **Exportación completa:** Puede exportar TODOS sus datos en CSV en cualquier momento, sin restricciones ni costos adicionales.

- **Backups diarios:** Mantenemos backups automáticos por 30 días. Si cancela el servicio, le entregamos todos sus datos en formato estándar dentro de 30 días.

- **Sin lock-in:** El sistema está diseñado para que usted tenga control total. No encerramos su información.

- **Transparencia:** Puede ver exactamente qué datos tiene el sistema y exportarlos cuando quiera.

**Garantía:** Si alguna vez necesita sus datos y no se los proporcionamos, le devolvemos el 100% del mantenimiento pagado.

---

**Objeción: "¿Qué pasa si dejo de pagar el mantenimiento?"**

**Respuesta:**
Entiendo la preocupación. Aquí está exactamente qué pasa:

- **Sus datos están seguros:** Aunque deje de pagar, sus datos NO se eliminan. Se mantienen por 90 días después del último pago.

- **Sistema sigue funcionando:** El sistema continúa operativo. Solo pierde acceso a:
  - Soporte técnico
  - Actualizaciones de seguridad
  - Backups automáticos
  - Mejoras menores

- **Reactivación:** Si decide volver, simplemente reanuda el pago mensual. No hay penalizaciones ni costos de reactivación.

- **Exportación final:** Si decide cancelar definitivamente, le entregamos todos sus datos en CSV dentro de 30 días, sin costo adicional.

**Recomendación:** El mantenimiento mensual es esencial para seguridad y actualizaciones. Pero si necesita pausar temporalmente, podemos discutir opciones.

## 7. Checklist para Cerrar Ventas

### 7.1 Antes de Enviar Propuesta
- [ ] Entender necesidades específicas del cliente
- [ ] Identificar paquete adecuado (Basic/Standard/Premium)
- [ ] Calcular precio final (dentro de rangos aceptables)

- [ ] Ofrecer valor, no descuentos

- [ ] Contrato de servicios claro (setup + mantenimiento)
- [ ] Pago de setup antes de comenzar (o 50% al inicio, 50% al finalizar)

- [ ] Obtener datos iniciales del cliente
- [ ] Establecer canales de comunicación

- [ ] Entregar sistema en tiempo acordado
- [ ] Capacitación según paquete
- [ ] Documentación entregada
- [ ] Seguimiento primera semana post go-live
